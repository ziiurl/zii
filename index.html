Introduction

Zii URL Shortener at zii.bz is a platform that takes long URLs and compresses them into fewer characters to make a URL link that is easiest to share, tweet, or email to friends & family.

Description

Users can create these short links through the web interface at zii.bz, or programatically create them through the URL Shortener API. With the Zii URL Shortener API you can write applications that use simple HTTP methods to create, inspect, and manage zii.bz short links from a desktop, mobile, or web. Links that users create through the URL Shortener can be automatically operated in your mobile applications that can handle those links, respectively. This automatic behavior provides the possible experience to your app users who open zii.bz links, no matter what platform or device they are on.

Quick start

Create a shortened URL

  • Visit the zii.bz URL shortener site at
  • If you aren’t signed in, click the Register button in the top right corner to get you started.
  • Write or paste your URL in the Generate short link box.
  • Click SUBMIT
    • Below the “Generate short link ” box, you’ll see the long version of your url and the short version. Once created, your link won’t expire unless it has to be removed for spam, privacy, or legal reasons.
